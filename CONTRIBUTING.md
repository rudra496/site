# Contributing to Rudra Sarker Portfolio

Thank you for your interest in contributing! This is a personal portfolio site, but suggestions and improvements are welcome.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/improvement`)
3. Make your changes
4. Commit with a descriptive message (`git commit -m 'Fix: description'`)
5. Push to your branch (`git push origin feature/improvement`)
6. Open a Pull Request

## Guidelines

- Keep the design consistent with the existing theme
- Optimize images before adding them
- Test changes locally before submitting
- Write clean, well-indented HTML/CSS/JS

## Reporting Issues

Found a bug or broken link? Please [open an issue](https://github.com/rudra496/site/issues) with:
- Description of the issue
- Steps to reproduce
- Browser and device information
